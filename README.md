# TFG_Un_sistema_d_alerta_primerenca
Codi usat per al Treball de Final de Grau "Anàlisi i Predicció del Risc de Fracàs Acadèmic: Un Sistema d'Alerta Primerenca". 
L'objectiu és detectar estudiants en risc de fracàs acadèmic a partir de les notes i els logs d'activitat d'una Aula Virtual.
